# Logos dos Times

⚠️ **IMPORTANTE**: Os SVGs dos times devem ser colocados na pasta `public/assets/teams/`, não nesta pasta.

## Estrutura

Os arquivos devem ser salvos em:
- `public/assets/teams/serie-a/` - Logos dos times da Série A
- `public/assets/teams/serie-b/` - Logos dos times da Série B

## Convenção de Nomes

1. Nome do time em minúsculas
2. Espaços substituídos por hífens
3. Sem acentos
4. Apenas letras, números e hífens

Exemplos:
- `flamengo.svg`
- `sao-paulo.svg`
- `atletico-mg.svg`

O sistema buscará automaticamente os logos baseado no nome do time quando criar uma rodada.
